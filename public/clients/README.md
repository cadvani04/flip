# Client Logos

Place client logos in this directory with the following filenames:

- `nailedit-labs.png` - NailedIt Labs
- `total-fire-protection.png` - Total Fire Protection
- `airbnb.png` - Airbnb Expansion Projects
- `onyx-lifestyle.png` - Onyx Lifestyle Club
- `chaga-company.png` - The Chaga Company
- `adaseco.png` - AdasEco
- `startup-ecosystem.png` - Founder & Startup Ecosystem Projects

If logos are not found, they will be automatically hidden and only the company name and sector will be displayed.
