# Team Member Images

Place team member photos in this directory with the following filenames:

- `iman-yael-schafer.jpg` - Iman Yael Schaefer
- `bee-shondi-schafer.jpg` - Bee Shondi Schafer  
- `curran-advani.jpg` - Curran Advani
- `sergio-pineros.jpg` - Sergio Pineros

If images are not found, the component will automatically use placeholder avatars generated from the team member names.
