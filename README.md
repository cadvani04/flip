# Flip Ecommerce Website

A modern, responsive website for Flip - a growth partner that transforms unsold inventory into profitable revenue streams.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Modern Design**: Clean, professional design with Tailwind CSS
- **Responsive Layout**: Fully responsive across all device sizes
- **Component-Based**: Built with React and Next.js for maintainability
- **Performance Optimized**: Built with Next.js 14 for optimal performance
